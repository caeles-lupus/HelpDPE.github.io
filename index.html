Для открытия программы нужно дважды щелкнуть на иконку программы на рабочем столе.
После открытия надо нажать «Файл» - «Открыть» для открытия изображения, либо «Файл» - «Создать» для создания нового.

Инструкция по использованию

Чтобы применить эффект к изображению нужно нажать на третью иконку сверху, затем выбрать нужный эффект. После выбора эффекта появится небольшое окно, на котором можно задать настройки для текущего эффекта. Когда настройки будут заданы, нужно нажать «ОК» для применения эффекта, либо «Отмена», если он не подошел.
Для коррекции цветов нужно нажать на четвертую иконку сверху. После действовать так же, как и с эффектами.
Чтобы добавить стикер на изображение, нужно нажать на пятую иконку сверху, затем нажать на нужный стикер. Когда стикер появится на изображении, его можно перемещать и изменять размер.
Для добавления текста нужно нажать на шестую иконку сверху. Затем ввести нужный текст под словами «Ввод текста». Ниже можно выбрать шрифт либо из списка для быстрого выбора, либо из выпадающего списка, размер текста, а также цвет. После всех настроек, необходимо нажать на кнопку «Добавить текст». Текст можно переместить и изменить размер. После этого нужно нажать на текст правой кнопкой мыши и выбрать пункт «Готово», если нужно оставить текст.
Для рисования нужно нажать на первую иконку. Затем выбрать инструмент: кисть, прямая, эллипс, прямоугольник. После этого нужно задать им цвет и размер. Когда все настройки заданы, левой кнопкой мыши можно рисовать по изображению.
